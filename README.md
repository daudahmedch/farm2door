Overview

FARM2DOOR is a simple web-based organic product delivery platform. It
allows users to view products, add them to a cart, place orders, and
track delivery progress.

1. Product Display

-   The homepage shows a list of available organic products.
-   Each product displays an image, name, category, price, and buttons
    to view or add to cart.

2. Search & Filter

-   Users can search products by name.
-   Users can filter products by category.
-   The product grid updates instantly based on search and filter
    options.

3. Add to Cart

-   Clicking “Add” adds the product to the shopping cart.
-   The cart updates automatically and uses localStorage for saving
    items.
-   Users can view items, remove items, and see total price.

4. Order Management

-   When the user confirms the cart, an order is created.
-   The order is saved in localStorage for persistence.

5. Delivery Tracking

-   Each order includes a simple delivery tracking flow:
    1.  Order Placed
    2.  Packed
    3.  Out for Delivery
    4.  Delivered
-   The status updates automatically using JavaScript timers.

6. Reviews System

-   Users can submit product reviews with rating and text.
-   Reviews are saved locally.

7. Analytics Dashboard

-   Shows total products, total orders, cart items, and total reviews.
-   Values are calculated dynamically.



8. Responsive Design

-   The layout adjusts for mobile, tablet, and desktop.
-   Built using HTML, CSS, and JavaScript.
